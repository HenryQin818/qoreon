# Code of Conduct

## Our standard

We want Qoreon to be a practical, respectful project for builders, operators, and AI workflow explorers.

Please:

- assume good intent
- be clear and specific
- focus on reproducible facts and user impact
- disagree on ideas without attacking people

## Unacceptable behavior

We do not accept:

- harassment or personal attacks
- repeated bad-faith arguments
- hate speech or discriminatory language
- publishing private credentials, session data, or internal-only traces

## Maintainer responsibilities

Maintainers may edit, hide, or remove comments, issues, pull requests, and other contributions that violate this policy.

## Reporting

If you find behavior that should be reviewed, open a private report through the repository owner or use another trusted channel to contact a maintainer.

## Scope

This code of conduct applies to issues, pull requests, discussions, and other public project spaces.
