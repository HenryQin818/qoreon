# 推荐结构边界与真源说明

## 真源

- 项目真源：`seed/project_seed.json`
- 通道真源：`seed/channels_seed.json`
- Agent 真源：`seed/agents_seed.json`
- 任务真源：`seed/tasks_seed.json`
- 技能真源：`seed/skills-manifest.json`

## 边界

- 可以公开的是 seed、skills、示例任务、公开知识文档
- 不可以公开的是真实 session、run、消息历史和私有路径

## 推荐结构

- 中央：总控 + 架构
- 执行：后端 + 前端 + 数据 + 测试
- 支撑：文档 + 用户镜像 + Git桥接
