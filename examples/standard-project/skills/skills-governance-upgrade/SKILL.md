# skills-governance-upgrade

## 用途

用于管理标准项目里的技能清单、技能升级、技能培训和公开安全校验。

## 适用场景

- 新增一个 skill
- 替换某个已有 skill
- 判断某个 skill 是否适合进入默认包
- 做技能使用培训

## 检查清单

1. 是否公开安全
2. 是否有明确职责边界
3. 是否写清输入、步骤、输出
4. 是否会误导为默认可写能力
5. 是否需要手工确认

## 输出

- 保留 / 不保留 结论
- 技能边界说明
- 培训口径

## 边界

- 不引入来源不明的外部 skill
- 不把私有路径、私有会话、私有运行态带入公开包
