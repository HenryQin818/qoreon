统一任务样式场景包 v3

当前场景包按最终口径收敛：通道页回到任务列表模式，顶部任务面板继续保留现有任务页的时间轴与横向窄子任务；三类场景的任务卡统一使用正式 task 页的五种状态、淡色状态底和现有 Agent 头像语法，底部 `主 / 管 / 协 / 验` 继续分组展示。
通道页列表模式 任务页时间轴 横向窄子任务 主 / 管 / 协 / 验 气泡分组 现有头像语法 五态样本已齐

本轮判断重点

以下四条是 `v3` 的判断重点，产品可以直接按这四条复看三个场景页。
  • 通道页中区回到列表模式，不再使用时间轴；顶部任务面板继续保留现有任务页的事件轴与横向窄子任务。
  • 任务卡顶部角色位不再单独占一行，改为和原先参与Agent 合并成底部一体展示区。
  • 底部 Agent 展示区统一按 `主 / 管 / 协 / 验` 分组展示头像，并直接沿用现有项目 Agent 头像的样式语法。
  • 三个场景都继续只聚焦任务项，并覆盖 `待办 / 进行中 / 待验收 / 已完成 / 暂缓` 五种正式状态样本，同时按状态给出淡色背景。

通道页任务列表

通道页中区回到列表模式，不再使用时间轴；新的任务板块内容直接放进现有任务卡语法里，状态与头像风格都和正式 task 页对齐。
