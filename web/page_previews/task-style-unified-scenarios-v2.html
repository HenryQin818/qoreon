统一任务样式场景包 v2

当前场景包按两条新增硬门禁重做：一是所有任务卡都把 `主 / 管 / 协 / 验` 核心协作位前置成单字气泡；二是 `通道页任务列表` 与 `顶部任务面板全量视图` 全部回归现网任务面板的时间轴布局基线，不再用普通列表或卡墙替代。
主 / 管 / 协 / 验 前置 通道页回到左总右子 任务面板回到左总右子 侧边栏窄栏同语法 五态样本已齐

本轮门禁核对

以下四条是 `v2` 的判断重点，产品可以直接按这四条复看三个场景页。
  • 所有任务卡第一眼都先看到角色气泡：`主 / 管 / 协 / 验`，不再把核心协作位藏在文案里。
  • 通道页与顶部任务面板都回到时间轴式主布局：左侧总任务，右侧子任务链。
  • 消息右侧任务侧边栏虽然更窄，但仍然沿用同一套角色气泡、正式状态、单条结论文案和参与Agent 表达。
  • 三个场景都继续只聚焦任务项，并覆盖 `待办 / 进行中 / 待验收 / 已完成 / 暂缓` 五种正式状态样本。

通道页时间轴任务面板

沿用 task 页通道结构，但中区回归时间轴布局：一条总任务在左，子任务链在右。
